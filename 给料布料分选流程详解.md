# 碲化铋颗粒给料→布料→检测→分选全流程详解

> 核心解答：玻璃盘上的"均匀分布"并非颗粒散落铺满盘面，而是**颗粒沿盘面某一固定半径排列成一圈**，像钟表的刻度一样均匀分布，逐一旋转经过检测工位。

---

## 一、流程总览

```
补料斗 → 精密振动盘(排队) → 传送轨道(单排输送) → 玻璃盘入料口(逐个上盘)
      → DDR转盘旋转(逐一过检) → 6相机工位(逐面拍摄+AI识别)
      → 分选工位(气动阀吹气剔除) → OK/NG/回收三个下料通道
```

---

## 二、逐步详解（重点：玻璃盘上的排列与分选机制）

### 📦 第1步：给料——从杂乱到排队

**设备**：补料斗 + 精密振动盘

| 动作 | 具体描述 |
|---|---|
| 补料 | 碲化铋颗粒倒入补料斗。当振动盘内物料低于阈值时，自动补料系统向振动盘补充物料，无需人工频繁加料 |
| 振动排列 | 精密振动盘启动，通过定向振动将杂乱堆叠的颗粒沿螺旋轨道"理顺"，逐个排列成单行输出。就像超市收银台排队——一堆人从门口进来，经过排队通道变成一列 |

**关键**：振动盘的出口是一条窄通道，宽度只允许一颗颗粒通过，因此输出的是**单排、等间距**的颗粒流。

---

### 📐 第2步：布料——颗粒如何上玻璃盘并"均匀分布"

**这是你最关心的环节。答案：颗粒不是铺满盘面，而是沿盘面边缘排列成一圈。**

**设备**：传送轨道 + 光学玻璃检测盘 + DDR直驱电机 + 低压吸盘

#### 2.1 颗粒上盘

振动盘输出的单排颗粒，沿传送轨道滑到玻璃盘的**入料口位置**（盘面边缘的某个固定点）。颗粒到达入料口时，被放置到玻璃盘的**边缘区域**——一个特定的半径圈上。

```
想象一个旋转木马：
- 木马的座位都排列在同一个圆周上（不是散落在整个圆盘上）
- 每个座位之间的间距是固定的
- 游客（颗粒）逐个上车，坐在外圈的固定位置
```

#### 2.2 低压吸盘固定位置（关键机制）

**任务书原文**："在转盘边部布置低压吸盘，保证所有粒子排列在转盘的某一固定半径处"

这就是核心——玻璃盘边缘有一圈**低压吸盘（真空吸嘴）**，当颗粒落入盘面时：
1. 颗粒被低压吸盘吸附在盘面边缘的**固定半径位置**
2. 所有颗粒都在同一个半径圈上排列，不会偏离、不会重叠
3. 吸力足够固定颗粒但不损伤脆性碲化铋材料（低压，不是强力真空）

```
           玻璃盘俯视图（简化示意）

                相机工位6
                    ↓
              ·  ·  ·  ·  ·  · ← 颗粒排列在固定半径圈上
            ·                  ·   （低压吸盘固定位置）
          ·      DDR转盘        ·
         ·     （玻璃盘）         ·  ← 转盘中心
          ·                      ·
            ·                  ·
              ·  ·  ·  ·  ·  ·
                    ↑
                相机工位1
                
          每个颗粒间距由进料速度和转盘转速控制
          DDR电机驱动整个盘面匀速旋转
          颗粒逐个经过6个相机检测工位
```

#### 2.3 间距控制——"视野内每次只出现一个粒子"

**任务书原文**："调整DDR转盘速度使视野内每次只出现一个粒子"

- DDR转盘的转速和振动盘的进料速率需要**精确匹配**
- 进料快 + 转盘慢 → 颗粒间距太近 → 相机视野内出现多个粒子 → 识别干扰
- 进料慢 + 转盘快 → 颗粒间距太大 → 检测效率降低
- 最佳状态：**颗粒间距恰好等于一个相机视野宽度**，旋转过程中每个视野只有一颗粒子

---

### 📸 第3步：检测——逐面拍摄 + AI识别

**设备**：6台工业相机 + 显微镜头 + 环形光源 + 频闪控制器 + 光纤传感器

#### 3.1 定位触发（PLC硬触发）

颗粒在传送轨道上移动时，**对射光电传感器**检测到颗粒到达 → 向PLC发送信号 → PLC计算颗粒抵达相机位置的时间 → PLC在精确时刻**硬触发**相机快门 → 相机触发频闪控制器闪光。

**为什么是PLC硬触发而非PC触发？**
- PC（工控机）计算有延迟波动，高速运转时几毫秒的偏差就会导致颗粒偏离拍摄窗口
- PLC是实时硬件控制器，计算时间确定性高，定位精度更可靠

#### 3.2 六面逐工位检测

玻璃盘旋转一圈 = 颗粒经过6个检测工位 = 6台相机各拍一面 = 360°全表面覆盖

```
玻璃盘旋转方向 →→→（顺时针）

  工位6          工位5          工位4
   ↓              ↓              ↓
  [相机6]       [相机5]       [相机4]
   拍第6面       拍第5面       拍第4面

     ·→→→→→→→→→→→→→→→→→·  ← 颗粒在盘面边缘旋转前进
  [相机1]       [相机2]       [相机3]
   ↑              ↑              ↑
  工位1          工位2          工位3
   拍第1面       拍第2面       拍第3面

  颗粒从入料口进入 → 经过工位1 → 工位2 → ... → 工位6
  每经过一个工位，该工位的相机+光源拍摄一个面
  6面全部拍摄完成后 → 进入分选工位
```

#### 3.3 AI识别

- 相机拍摄的图像传入工控机
- OpenCV动态优化图像参数 → 边缘提取、轮廓分割
- VGG/ResNet深度学习模型识别缺陷类型（镀层面缺损、掉镀层、脏污、开裂）
- 单帧处理≤100ms → 识别结果标记为OK / NG / Undetected（置信度低）
- 识别结果通过TCP/IP传回PLC

---

### 💨 第4步：分选——气动阀吹气剔除

**设备**：高频电磁阀 + 压缩空气系统 + 下料通道

**这是另一个容易困惑的点——分选不是"从盘上取走"，而是"吹气推离"。**

#### 4.1 分选机制

颗粒完成6面检测后，继续随玻璃盘旋转到达**分选工位**（紧接检测工位之后的位置）。分选工位旁边设有压缩空气喷嘴，对应不同下料通道。

PLC已经知道：
- 这个颗粒在盘面上的**位置**（由PLC自己计算，从入料口到分选工位经过的角度是固定的）
- 这个颗粒的**识别结果**（AI模型传回的OK/NG/Undetected标记）

PLC在颗粒到达分选工位时，根据识别结果触发对应的高频电磁阀：

| 识别结果 | 电磁阀动作 | 颗粒去向 |
|---|---|---|
| ✅ OK | 喷吹入OK通道 | 合格品收集区 |
| ❌ NG | 喷吹入NG通道 | 废料区 |
| 🔄 Undetected | 喷吹入回收通道 | 人工复检区 |

```
分选工位侧视图（简化示意）

                    ← 压缩空气喷嘴方向
                    ↓
    OK通道 ← [喷嘴OK]    ═══════  ← 玻璃盘边缘（颗粒在此经过）
    NG通道 ← [喷嘴NG]    
    回收通道 ← [喷嘴回收]    

    颗粒随盘面旋转到达分选工位时：
    - OK颗粒：对应OK喷嘴吹气，将颗粒吹入OK合格品通道
    - NG颗粒：对应NG喷嘴吹气，将颗粒吹离盘面落入NG废品通道
    - Undetected颗粒：对应回收喷嘴吹气，吹入回收通道待人工复检
    ⚠️ 注意：所有颗粒都通过喷吹脱离盘面，真空腔是整圈环形密封结构，无法单独控制每个点位
```

#### 4.2 为什么用气吹而非机械剔除？

- 碲化铋颗粒**质地脆、表面有功能镀层**
- 机械剔除（推杆、夹取）容易损伤颗粒或破坏镀层
- 气吹是柔性接触——压缩空气的压力和方向可精确控制，既能推走颗粒又不损伤
- 高频电磁阀响应速度极快（毫秒级），匹配≥300 PCS/min的分选速率

#### 4.3 分选后的颗粒去哪？

- 低压吸盘**全程持续吸附**（真空腔为环形密封结构，无法单独控制每个点位）
- OK颗粒：喷吹克服低压吸附力，将颗粒吹入OK合格品通道
- NG/Undetected颗粒：对应喷嘴吹气，将颗粒吹入NG废品通道或回收复检通道
- 玻璃盘继续旋转，空出的位置回到入料口接收新颗粒

## 三、完整时序流程（一颗颗粒的一生）

```
时间轴 →

t₀  补料斗自动补充 → 振动盘振动排列 → 颗粒排队等待
t₁  颗粒从振动盘出口滑出 → 进入传送轨道
t₂  光纤传感器检测到颗粒到达 → PLC记录"颗粒A入盘"
t₃  颗粒落入玻璃盘边缘 → 低压吸盘吸附固定在固定半径位置
t₄  DDR转盘旋转 → 颗粒A经过工位1 → PLC硬触发相机1 → 拍摄面1
t₅  继续旋转 → 经过工位2 → PLC硬触发相机2 → 拍摄面2
t₆  ...经过工位3、4、5、6 → 拍摄面3、4、5、6
t₇  6面图像全部传入工控机 → AI模型识别 → 结果传回PLC
t₈  颗粒A旋转到分选工位 → PLC根据识别结果控制电磁阀
    - OK：OK喷嘴吹气，颗粒被吹入OK合格品通道
    - NG：NG喷嘴吹气，颗粒被吹入NG废品通道
    - Undetected：回收喷嘴吹气，吹入回收复检通道
t₉  空位回到入料口 → 接收下一颗颗粒 → 循环继续
```

**关键时间约束**：
- 从入盘到分选完成 = 玻璃盘旋转一圈的时间
- 盘面同时有多个颗粒在不同工位 → 形成流水线效应
- 玻璃盘不停顿，连续旋转 → 实现≥300 PCS/min的连续吞吐

## 四、为什么是"一圈排列"而非"铺满盘面"

| 方案 | 颗粒分布 | 问题 |
|---|---|---|
| ❌ 铺满盘面 | 颗粒散落在整个盘面各处 | 相机视野内可能同时出现多颗 → 识别干扰；颗粒间距不可控 → 无法精准定位分选 |
| ✅ 固定半径一圈排列 | 所有颗粒在同一半径上等间距排列 | 每个视野只有一颗 → 识别清晰；PLC可精确计算每颗颗粒位置 → 分选定位精准 |

**类比**：这就像工厂流水线 vs 杂乱堆放。流水线上工件等间距移动，每个工位只处理一件；杂乱堆放则无法控制处理顺序和精度。

---

## 五、流水线效应（盘面上同时有多颗颗粒）

实际运行中，玻璃盘面上**同时存在多颗颗粒**，它们在不同工位：

```
盘面上某一时刻的状态（6颗颗粒同时在盘上）：

          颗粒F(待检测面6)  颗粒E(待检测面5)  颗粒D(待检测面4)
               ↓                ↓                ↓
             [工位6]          [工位5]          [工位4]

               ·→→→→→→→→→→→→→→→→·→→→→→→→→→→→→·→→→→→→→→→→→→·

          颗粒A(刚入盘)    颗粒B(待检测面1)  颗粒C(待检测面2)
               ↑                ↑                ↑
             [入料口]          [工位1]          [工位3]

  同时：
  - 颗粒G刚完成6面检测，正在分选工位被气吹剔除
  - 颗粒A刚入盘，等待旋转到工位1
  → 盘面形成"流水线"，连续吞吐
```

这就是为什么能实现≥300 PCS/min——不需要一颗颗粒走完全程再处理下一颗，而是多颗同时在不同阶段，形成连续流水线。
